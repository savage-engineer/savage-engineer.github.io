Open Source

During the first year of my degree we were given a couple of open-ended projects, some of them working in groups. One such project developed intoCoffee School, an interactive set of lessons teaching introductory JavaScript. These run in-browser and culminate in the student creating a flappy-bird style game prototype.

The code for the lessons and the platform they run on is up onGitHub.

Hackathons!

Over the past year I've been to a fair few hackathons, from BrumHack and BathHack to hackCambridge and HackLondon. Not all of these events yielded a completed "thing". From trying out new technologies to building something cool, just because, I've found that I learned a lot during the past year.

Things made range fromStockstroids, an asteroids prototype where the game logic is determined by data from Bloomberg's API, to weird hacks involving an Intel Edison, VB, Windows 95/98 and translation into morsecode (all at once!).
